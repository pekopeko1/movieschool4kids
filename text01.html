はじめての動画へんしゅう

小学3年生のみんな、こんにちは！
今日は、動画をへんしゅうする「まほう」をいっしょに学んでいこう！✨

はじめに：動画へんしゅうって なんだろう？

🎉 こんにちは！今日はみんなで、動画へんしゅうにチャレンジするよ！

動画へんしゅうは、動画を「もっとおもしろく」「もっとわかりやすく」するためのまほうなんだ！

🌟 今日のゴール：
3つのまほう（カット、テロップ、BGM）をおぼえて、自分だけの30びょう動画をつくるじゅんびをしよう！

さあ、いっしょにたのしもう！✨

準備しよう：まほうの道具をひらこう🧰

  • つかう道具は「Clipchamp」だよ！
  • インターネットで「clipchamp.com」をひらこう。
  • 「新しいビデオを作成」ボタンをおすよ。
  • 「メディアのインポート」ボタンをおして、デスクトップにある「sample.mp4」をえらぼう！
  • 読みこんだ動画は「タイムライン（時間じく）」にドラッグ＆ドロップ！

魔法その１：カット✂️（いらない部分をけずる魔法）

📌 なんのためにやるの？
→ まちがえたところや、つまらない部分をなくして、見やすくするため！

📌 つかうタイミングは？
→ 「言いまちがえた！」「このへんちょっとながいなぁ」そんな時！

やってみよう！

  1. いらない部分の「はじまり」に白い線を合わせる
  2. ハサミマークの「分割」ボタンをおす
  3. 「おわり」にも白い線を合わせて、また「分割」ボタン！
  4. まんなかのいらない部分をクリック→「削除」ボタンをおす！
  5. できたすきまを、ドラッグしてぴったりつけよう！

魔法その２：テロップ✍️（もじを入れる魔法）

📌 なんのためにやるの？
→ もっとわかりやすく！気持ちをつたえるため！

📌 どんなときに？
→ 「すごい！」「みてみて！」って言いたいとき、しゃべったことを文字で見せたいとき

やってみよう！

  1. 左のメニューから「テキスト」をえらぶ
  2. 好きなデザインをタイムラインにドラッグ！
  3. テロップのながさをちょうせいしよう
  4. 右がわで、言葉・色・大きさを変えられるよ！

魔法その３：BGM🎵（おんがくを入れる魔法）

📌 なんのために？
→ ふんいきをもりあげて、もっと楽しい動画に！

📌 どんなときに？
→ 明るい・かっこいい動画にはぴったりの音楽をつけよう！

やってみよう！

  1. 左のメニューから「音楽とサウンドエフェクト」をえらぶ
  2. 好きな音楽をタイムラインにドラッグ！
  3. ながすぎたらカットの魔法で短くしよう
  4. 右がわの「オーディオ」で音の大きさをちょうせい！

まとめと宿題：きみだけの30秒動画をつくろう！

🎯 今日のおさらい！

  • カット：いらない部分をけずる！
  • テロップ：もじをいれる！
  • BGM：音楽でふんいきアップ！

🏠 宿題！
「sample.mp4」をつかって、30びょうの動画をつくろう！

→ カット ✂️ テロップ ✍️ BGM 🎵 をつかってみよう！

✨ 完成したら、「エクスポート」→「1080p」で保存！
おうちの人といっしょにやってみてね！

付録：Clipchampのアカウント作成方法（保護者の方へ）

Clipchampのご利用にはMicrosoftアカウントでのサインインが必要です。

  1. clipchamp.com にアクセス
  2. 「無料で試す」をクリック
  3. Microsoftアカウントでサインイン、またはアカウントを新規作成

ご家庭で事前のご準備をお願いいたします。

用語集（わからなくなったらここをみてね！）

  • カット：動画のいらない部分を切りとること
  • テロップ：動画にいれる文字のこと
  • BGM：動画の後ろで流す音楽（バックグラウンドミュージック）
  • インポート：パソコンの中のデータをソフトに読みこむこと
  • タイムライン（時間じく）：時間にそって動画をならべるところ
  • 分割（ぶんかつ）：1つの動画を2つにわけること
  • エクスポート（書き出し）：できあがった動画をファイルにすること
